    \textbf{Abstract}

The issue of reward balancing in game economy design is central to a game's success. An unbalanced reward system can lead to decreased player engagement, unnaturally accelerated game progress, and can impact the overall sustainability of the in-game economy. Several previous studies have discussed game economic systems in general, but these approaches have focused on the macro scale and have not considered the dynamics of player behavior. This research aims to develop a reward system optimization that can account for variations in player behavior using an evolutionary algorithm integrated with agent simulation. Each agent is used to represent the dynamics of player behavior interacting within the game environment. The evolutionary algorithm is used to find the optimal combination of reward parameters that results in the most balanced system. By emphasizing a computational-mathematical approach, this research is expected to produce a balanced and sustainable reward system configuration across a variety of player scenarios while also contributing to the scientific design of dynamic and change-resistant game economic systems.
